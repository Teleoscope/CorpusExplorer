# Resources and Using Teleoscope
Teleoscope is a system for exploring large document sets by building machine learning workflows.
Teleoscope (from [teleology](https://en.wikipedia.org/wiki/Teleology) and [telescope](https://en.wikipedia.org/wiki/Telescope)) helps to discover purpose and meaning.
The system is designed with qualitative researchers in mind, but can be used by anyone interested in exploring [themes](https://en.wikipedia.org/wiki/Thematic_analysis) in a large dataset.

Teleoscope is a simple but powerful system. We have provided a variety of resources to help you get to know Teleoscope, and how it can apply to your projects.

- [How-To Guides](/resources/tutorial): Explanations of how to use Teleoscope functionality to organize your data in powerful ways.
- [Examples](/resources/examples): Worked examples of Teleoscope projects.
- [Concepts](/resources/concepts): Guides to the concepts behind Teleoscope.
- [Reference](/resources/reference): Definitions of Teleoscope interface concepts and items.
