# Teleoscope Preprint Paper
Teleoscope was developed as part of [Dr. Paul Bucci's](https://paulbucci.ca) PhD project while working with [Dr. Ivan Beschastnikh](https://www.cs.ubc.ca/~bestchai/) in [UBC's Systopia Lab](https://systopia.cs.ubc.ca/).
You can read our academic publication about Teleoscope as a preprint here: [Arxiv Preprint](https://arxiv.org/abs/2402.06124).

If you end up using Teleoscope in your academic work, please cite our paper using the following [Bibtex](https://www.bibtex.org/):

```
@article{bucci2024teleoscope,
  title={Teleoscope: Exploring Themes in Large Document Sets By Example},
  author={Bucci, Paul and Foord-Kelcey, Leo and Lee, Patrick Yung Kang and Singh, Alamjeet and Beschastnikh, Ivan},
  journal={arXiv preprint arXiv:2402.06124},
  year={2024}
}
```

Or you can use the following Vancouver style citation:

```
Bucci P, Foord-Kelcey L, Lee PY, Singh A, Beschastnikh I. 
Teleoscope: Exploring Themes in Large Document Sets By Example. 
arXiv preprint arXiv:2402.06124. 2024 Feb 9.
```
